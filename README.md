## 本地开发

### node版本
node version >= 13.8

### 本地开发开启SSR模式
**nodemon shoudle be installed globaly**
1. npm run build:server
2. npm run build:client:modern
3. nodemon --inspect ./server_app/bundle.js


### 本地开发开启CSR模式
npm run start:client
